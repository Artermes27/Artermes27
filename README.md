### [Cadet Management System](https://github.com/Artermes27/cadet-management-app)
**PHP · MySQL · JavaScript · Apache**

A full-stack web application digitising the administration of a school cadet unit — parade scheduling, lesson planning, equipment logistics, and attendance tracking. Built as a solo A-level Computer Science project covering the full development lifecycle: requirements gathering, ER diagrams, system design, implementation, testing, and deployment on a Linux server.

- Event approval workflow with real-time colour-coded calendar

- 224-page technical write-up included in the repository

### [Wix Minecraft Bot](https://github.com/Artermes27/wix_minecraft_bot)
**Python · pyautogui · Wix**

End-to-end order fulfilment automation for a small Hypixel Skyblock coin business. The bot monitors a live Wix storefront, parses order exports, and drives Minecraft through the full delivery sequence (party → warp → trade/auction → screenshot → archive) using GUI automation — no game API, pure screen control.

- Deployed and handling real orders for 4 months (Jan–Apr 2022)
- Reads game chat logs to verify in-game events and branch retry logic
- Separate AFK farming bot with auto-reconnect and launcher restart cycles
